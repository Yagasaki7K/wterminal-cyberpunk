# Windows Terminal Cyberpunk Theme

A Windows Terminal Theme based in Cyberpunk 2077 (Samurai)

# How it works?

Just made a **Cyberpunk 2077 terminal theme**. The code is in `settings.json` just go to this location `C:\Users\YOURUSERNAME\AppData\Local\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\LocalState` and paste the code inside settings.json. If you has a previous settings, just change the `schemes` and settings in `list`.

One alternative to find the settings is opening your terminal, taping `Ctrl + ,` and then click on `Open JSON file`.

And GIF file are here: [samurai.gif](./samurai.gif). You need to put in this location, same than settings.json: `C:/Users/YOURUSERNAME/AppData/Local/Packages/Microsoft.WindowsTerminal_8wekyb3d8bbwe/LocalState/samurai.gif`

Copy and paste these files to the same folder where is `settings.json` located.

If the text is unreadable add this line (backgroundImageOpacity" : 0.4) to the code for PowerShell, Command Prompt, WSL, etc.
